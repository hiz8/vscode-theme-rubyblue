# Ruby Blue Theme

Dark, high contrast theme for Visual Studio Code.

## Install

Install from the [Visual Studio Code Marketplace](https://marketplace.visualstudio.com/items/hirofumii.rubyblue-theme).

After installation, press `Ctrl+K Ctrl+T` (or `Cmd+K Cmd+T` on macOS) and select "Ruby Blue".

## Release Notes

### Version 3.0.0

- **Major refactoring to modern VS Code theme standards**
- Updated minimum VS Code version to ^1.80.0
- Renamed theme file to follow modern naming convention (-color-theme.json suffix)
- Added comprehensive UI color tokens
- Added semantic highlighting support with semanticTokenColors
- Enhanced support for:
  - Activity Bar, Side Bar, Status Bar, Title Bar
  - Peek view, breadcrumbs, and editor widgets
  - Symbol icons and problem indicators
- Updated workspace configuration to modern standards
- Improved accessibility and high contrast support

### Version 2.1.0

- Update theme to fix indent guide being purple.
- Update icon.
