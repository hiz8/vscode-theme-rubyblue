# Ruby Blue Theme

Dark, high contrast theme for Visual Studio Code.


## Install

1. Open VS Code
2. Press `Ctrl+Shift+X` (or `Cmd+Shift+X` on macOS) to open the Extensions view
3. Search for `RubyBlue Theme`
4. Click Install
5. Press `Ctrl+K Ctrl+T` (or `Cmd+K Cmd+T` on macOS) to open the theme picker
6. Select "Ruby Blue" from the list

Alternatively, you can install it from the command palette (`Ctrl+Shift+P` or `Cmd+Shift+P`):
- Type `Extensions: Install Extensions`
- Search for `RubyBlue Theme`
- Click Install

- Added semantic highlighting support with semanticTokenColors
- Enhanced support for:
  - Activity Bar, Side Bar, Status Bar, Title Bar
  - Terminal with full ANSI color support
  - Git decorations and diff editor
  - Peek view, breadcrumbs, and editor widgets
  - Symbol icons and problem indicators
- Removed legacy theme files (.tmTheme.old, .less files)
